# 🪨📜✂️ Pedra, Papel e Tesoura (CLI)

Um jogo simples de Pedra, Papel e Tesoura rodando no terminal usando Node.js.

## 🧪 Objetivo

Projeto simples feito para aprendizado e prática com Node.js e lógica básica.

---

## 🚀 Sobre

Este projeto foi criado com o objetivo de praticar:

* Entrada e saída no terminal (CLI)
* Lógica de jogo
* Uso do módulo `readline` do Node.js

---

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/ItaloFSousa/pedra-papel-tesousa
```

2. Acesse a pasta:

```
cd pedra-papel-tesousa
```

3. Execute o arquivo:

```
node index.js
```

---

## 🎮 Como jogar

* Escolha **1** para iniciar o jogo
* Selecione sua jogada:

  * 🪨 Pedra
  * 📜 Papel
  * ✂️ Tesoura
* O bot escolherá uma opção aleatória
* O resultado será exibido no terminal

---

## 📌 Regras

* Pedra vence Tesoura
* Tesoura vence Papel
* Papel vence Pedra
* Escolhas iguais resultam em empate
